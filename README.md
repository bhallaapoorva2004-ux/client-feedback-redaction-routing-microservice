# Sentinel Guard Compliance Microservice

A high-performance full-stack compliance microservice that automatically redacts Personally Identifiable Information (PII) such as credit cards, phone numbers, and emails, and routes feedback based on automated sentiment analysis rules.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npm run dev
   ```
